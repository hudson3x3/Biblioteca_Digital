* Caso não haja Serviço de Aplicação API MeuMundoAvonAPI, criar.

Trocar chaves e urls de MktPlace para Produção 
(mktPlace_appId,mktPlace_tokenCatalog,mktPlace_ParticipantInsert,mktPlace_clientId,mktPlace_profileId,mktPlace_campaignId)

Trocar chaves e urls de Avon para Produção 
(userIdAvon,passAvon,devkeyAvon,urlLoginAvon,urlGetProfileAvon)

Trocar chave 'urlSite' para url de produção (ex. http://hml.meumundoavon.com.br)

Admin

Azure

* Criar um 'Diretório Virtual' dentro do Serviço de Aplicativo (MeuMundoAvon) 'Site' com nome de 'Admin'

* Criar Blob de produção (Conta de Armazenamento)
	Cruar Contêiner 'file' (política de acesso - Particular)
	Criar Contêiner 'ltm'	(política de acesso - Contêiner (público read))

* Replicar conteúdo do Blob de HML (https://meumundoavonhml.blob.core.windows.net) para produção (conteiners file e ltm)
